Public Instance methods

body()

Returns the body of the issue. This attribute is not loaded by default because it requires an extra call to the Sifter website.

update(updates)

Passed a hash with symbols as the keys, this method will update the issue. The only attributes editable are: priority, category, status, assignee (or assigned_to), subject, and body. When creating a comment, this method is used if the comment updates any of the above mentioned attributes.
