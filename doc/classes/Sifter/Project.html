issue(criteria_or_other)

Load one of the project‘s issues based on criteria. The criteria can be a hash of attributes and their values, the id of the issue, the number of the issue, or the subject of the issue.

issues(reload = false)

Load this project‘s issues and put them into the issues instance variable, unless they have already been loaded. Pass true as the first argument to reload the issues.

reload_issues!()

Reload this project‘s issues.
